Описание алгоритмов

При игровом действии поля имеют одинаковую логику сдвига, поэтому нужна функция, которая сдвигает плитки по одному направлению. Данный функционал реализован в методе int game2048_4D::compose(int &A0, int &A1, int &A2), Который также подсчитывает заработанные очки.

Метод void game2048_4D::keyPressEvent(QKeyEvent* pe) обрабатывает нажатия клавиш: применяет метод compose, подсчитывает очки, отображает их, вызывает обновление экрана.

При обновлении игрового поля данные из модели отображаются на экран. Если установлен флаг add, то на заранее подсчитаном количестве пустых клеток, случайно добавляется новый элемент.
